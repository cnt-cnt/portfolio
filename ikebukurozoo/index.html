作品について

池袋動物園(架空)の公式サイト制作。トップページのみ制作。
企画にあたり、いくつかの実際にある動物園のサイトを参考にした。特に、京都市動物園とふれあい動物園 ピクニカ共和国のサイトを参考にしている。
動物園の中でも教育系のものなのかデート向けか、都心にあるのか地方にあるのかなど詳細を決めていくのに時間がかかった。

池袋動物園について

池袋駅より徒歩10分 / 都心型動物園 / 小学生向け体験型 / 教育系 / 平日は校外学習の生徒がメイン / 週末はファミリー層がメイン

ターゲット

小学校 / 小学生 / ファミリー層

デザイン

小学生向けの体験型動物園を表現できるように、親しみやすく明るく楽しい雰囲気になるようイメージした。メインカラーはオレンジ、黄、黄緑の3色を使用している。色彩効果について調べていた際、オレンジは明るさや元気、黄色は愉快や軽快、黄緑は希望、好奇心を連想させるとのことでこの色にした。また、目がチカチカしないように少し色のトーンを落とした。

苦労、重視した点

SPサイトでのハンバーガーメニューとナビゲーションのjQueryの考え方が少し混乱した。
コーディングではインデントを意識し、見返しやすいコードになるようにした。またclass名の付け方を重視し、「 _ 」を用いて単語を区切った。class名は親要素を継承し、「親要素名_inner」や「親要素名_子要素名」となるようにした。
画像はPNGをWebpに変換し、軽量化を行った。また、定期的にHTMLエラーチェッカーで確認した。
